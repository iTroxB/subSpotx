# **subSpotx**

![](/img/sub1.png)

Subdomain discovery tool developed in Go.

## Install tool

```shell
sudo rm -rf /usr/bin/subSpotx
rm -rf $HOME/go/bin/subSpotx
go install github.com/iTroxB/subSpotx/cmd/subSpotx@latest
sudo ln -s $HOME/go/bin/subSpotx /usr/bin/subSpotx
```

## Use tool

- subSpotx -h
